# Task 1 – GTM Event Tracking Strategy

Designed a complete event tracking strategy for OrthoNow using **Google Tag Manager (GTM)** and **Google Analytics 4 (GA4)**.

### Deliverables

- Complete GTM Event Schema
- Booking Funnel Tracking
- dataLayer JSON Events
- GTM Trigger Configuration
- GA4 Funnel Exploration
- Google Ads Conversion Strategy
- Implementation Notes

# Task 2 – Consultation Landing Page

Developed a responsive healthcare landing page using **HTML**, **CSS**, and **Vanilla JavaScript**.

### Features

- Mobile-first responsive design
- Semantic HTML5
- Accessible form components
- Hero section with consultation form
- Trust indicators
- Benefits section
- Call-to-Action section
- Client-side form validation
- GTM `window.dataLayer.push()` implementation
- Thank-you state without page reload

- Vanilla JavaScript
